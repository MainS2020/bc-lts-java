The PKIX API contains packages for generation and processing of:
  • Version 1 and Version 3 X.509 certificates, Version 2 CRLs, and PKCS12 files.
  • Version 2 X.509 attribute certificates.
  • S/MIME and CMS (PKCS7/RFC 3852).
  • OCSP (RFC 2560).
  • Time Stamp Protocol (TSP): RFC 3161 and RFC 5544.
  • Evidence Record Syntax (ERS): RFC 4998.
  • CMP and CRMF: RFC 4210 and RFC 4211.
  • OpenPGP (RFC 4880).
  • Extended Access Control (EAC).
  • Data Validation and Certification Server (DVCS) - RFC 3029.
  • DNS-based Authentication of Named Entities (DANE).
  • RFC 7030 Enrollment over Secure Transport (EST).
